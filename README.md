# New Tab - 你的专属新标签页

这是一个超好用的 Chrome 新标签页扩展。不仅能帮你整理常用网站、多引擎快速搜索，还能一键把浏览器书签变成好看的快捷图标。界面支持高度自定义，让你的浏览器主页既实用又好看。

## ✨ 核心功能

### 🔖 网站快捷方式
- **随心排列**：拖拽就能调整顺序，怎么顺手怎么来。
- **分组管理**：支持文件夹嵌套，网站再多也能整理得井井有条。
- **批量操作**：支持一键批量删除或移动到其他页面，省时省力。
- **自动图标**：自动帮你抓取网站的小图标并存起来，告别光秃秃的文字链接。
- **排版随心**：可以选择固定位置，也可以让图标自动排满。

### 🔍 聚合搜索
- **分类检索**：内置常用、影视、阅读、资料等多种分类，找什么都快。
- **自己定义**：支持添加你常用的任何搜索引擎。
- **一键切换**：点击就能快速换个引擎搜，再也不用复制来复制去了。

### 📚 智能书签导入
- **一键搬家**：点一下就能把浏览器书签全部导入到主页上。
- **结构不变**：导入后依然保留你原有的文件夹分类。
- **自动美化**：自动为没有图标的网页配上好看的随机颜色。
- **聪明去重**：自动识别重复书签，多次导入也不会乱。
- **实时同步**：开启后，浏览器里新加的书签会自动同步到新标签页。

### 🎨 高颜值界面定制
- **精美壁纸**：支持每日必应壁纸，也可以上传自己喜欢的照片。
- **护眼模式**：支持深色模式，还能设置晚上 7 点到早上 6 点自动变黑，保护视力。
- **细节可调**：图标多大、圆角多少、间距多宽，统统由你决定。
- **时钟组件**：自带时间显示，位置还能随意挪动。

### 🧩 实用小工具
- **侧边栏设计**：左侧可以放置今日热榜和便签本，主搜索框会自动向右挪出空间。
- **随手便签**：随时记录灵感，支持多张便签，自动本地保存，备份时也不易丢失。
- **今日热榜**：内置微博、百度、GitHub、ReadHub、掘金等热搜，支持自己添加数据源、改名和排序。
- **网易云音乐**：听歌小助手。支持登录读取你的每日推荐，自带播放器。
- **智能避让**：如果屏幕太小，小组件会自动隐藏，不挡住你的常用网站。

### ☁️ 安全的数据同步
- **云端同步**：支持 WebDAV 备份，数据存在你自己的网盘里，安全放心。
- **解决跨域**：提供 Cloudflare Worker 方案，轻松解决部分网盘在浏览器里的跨域限制。
- **本地备份**：支持一键导出和导入配置文件，换电脑也能秒恢复。

## 🚀 快速开始

### 如何安装
1. 下载本项目并解压到本地。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 把右上角的**“开发者模式”**开关打开。
4. 点击左上角的**“加载已解压的扩展程序”**按钮。
5. 选择你刚刚解压出来的文件夹里的 `src` 目录即可。

### 怎么导入书签
1. 打开新标签页，点击右下角的**设置按钮（⚙️）**。
2. 找到**“书签导入”**选项卡。
3. 选好你想导入的书签范围（比如“书签栏”或“所有书签”）。
4. 点击**“⚡ 一键自动导入所有书签”**，瞬间完成。

> 💡 **小贴士**：
> 如果你在设置里开启了“启用自动同步”，以后只要你在浏览器里收藏了新网页，新标签页就会自动同步更新。

### 怎么备份和同步数据
1. 点击设置面板，切换到**“数据备份”**。
2. 填入你的 WebDAV 网盘地址、账号和密码（例如坚果云）。
3. 默认会直接连接网盘。如果因为网盘限制连不上，可以配合自己部署的 Worker 中转使用。
4. 点击**“验证连接”**，显示成功后就可以放心上传和下载配置了。

> ⚠️ **安全提醒**：
> 请不要随便使用别人提供的陌生中转服务，因为中转过程可能会接触到你的网盘密码。如果使用自己部署的 Worker，建议设置一个访问密码（Token）来保护它。

### 🛠️ 部署自己的 WebDAV 中转（可选）
如果你的网盘在浏览器里直接连不上，可以几步部署一个专属的 CF Worker 中转：

1. 进入 `worker` 目录，运行部署命令：
   ```bash
   cd worker
   wrangler deploy
   ```
2. 部署成功后，把中转地址填入扩展设置即可：
   ```text
   https://newtab-webdav-proxy.你的账号.workers.dev
   ```

为了防范他人滥用，建议加上安全锁：
- **设置访问令牌 (Token)**：
  ```bash
  wrangler secret put PROXY_TOKEN
  ```
  设置后，在扩展的“中转地址”后面加上参数即可：`https://...workers.dev/?token=你的令牌`。
- **限制只允许访问你的网盘域名**：
  ```bash
  wrangler secret put ALLOWED_WEBDAV_HOSTS
  ```
  值填写你的网盘域名，如 `dav.jianguoyun.com`。

## 📁 项目结构

这是一个干净清爽的项目，主要包含以下内容：
- `src/`：扩展的核心代码（日常使用只需加载这个文件夹）。
- `worker/`：WebDAV 中转 Worker 的部署代码（如果不需要云同步中转，可以忽略）。

- 纯 HTML / CSS / JavaScript
- Chrome Extension API
- Chrome Bookmarks API
- localStorage 数据存储
- WebDAV 云端同步（可选）
- Cloudflare Worker WebDAV 中转（可选）

## 🔐 关于安全的小建议

- **优先直连**：扩展默认会直接连接你的网盘。只有在直连失败（比如跨域限制）时，才需要考虑部署 Worker 中转。
- **保护好网盘密码**：因为中转服务会帮你传递数据，所以**强烈建议使用自己部署的 Worker**，不要使用来路不明的第三方中转。
- **给 Worker 加上锁**：为了防止别人蹭用你的 Worker 流量，建议在部署时设置 `PROXY_TOKEN`（访问令牌）和 `ALLOWED_WEBDAV_HOSTS`（网盘白名单）。这样既安全，又省心。
